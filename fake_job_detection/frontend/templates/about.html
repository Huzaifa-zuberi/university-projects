About This Project

Fighting online job fraud in Pakistan with machine learning.

Project Overview

Detecting fake job postings

JobShield is an academic project that predicts whether a job advertisement is genuine or a scam. It studies the wording of the posting using a trained machine learning model and gives a clear, easy-to-understand result.

  • Built with HTML, CSS, JavaScript and Flask.
  • Machine learning powered by Scikit-Learn (TF-IDF + classifier).
  • Compares Logistic Regression, KNN, and SVM.
  • Records stored in PostgreSQL (Supabase).

Important notice

This is an awareness and learning tool. A "Real" result does not guarantee a job is genuine. Always stay cautious, never pay fees, and verify the company through official channels before sharing any personal details.
